Include the script tag below in your document and get start! 
<script type='text/javascript' src='js/monster-dropdown.js'></script>

Sample 1

This is the code needed to achieve this: 
$('#dropdown').monsterDropdown();

Sample 2

This is the code needed to achieve that: 
$('#dropdown2').monsterDropdown({display:'top',
                            height:200,
                            callback:onDropdownChange});
                            
function onDropdownChange(val){
    $('.callbackOutput').text("Selected : "+ val);
}

Sample 3 with jScrollPane

Make sure to include jScrollPane css & js and below is the code needed to achieve with jScrollPane: 
$('#dropdown3').monsterDropdown({height:200});
                            
//jScrollPane only work when target DIV is visible                          
$('#dropdown3').parent().find('.optionsWrapper').show();
$('#dropdown3').parent().find('.optionsWrapper').jScrollPane();
$('#dropdown3').parent().find('.optionsWrapper').hide();

Usage

Understands default options that can use for this plugin. 
$('#dropdown').monsterDropdown({
    display:'top', //display option list on 'top' or 'bottom', empty/'auto' will auto display
    height:0, //set height for option list, empty/0 is auto height
    callback:'', //call back function
});

//Option can be reset once the same element is init.
$('#dropdown').monsterDropdown('reset');

//Option can be update later once the same element is init.
$('#dropdown').monsterDropdown('update','display','top');
