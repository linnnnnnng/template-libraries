How to install

First and most important, the jQuery library needs to be included, then include the script tag below in your document and get start! 
<script type='text/javascript' src='js/monster-pagination.js'></script>
DOWNLOAD monster-pagination.js

Sample 1

This is the code needed to achieve this:
Note that the call must be made inside of a $(document).ready() call, or the plugin will not work! 
$(document).ready(function(){
    $('#pagination').monsterPaginate({
        totalPages:12,
        display:5,
        showFirstLast:false
    });
});

Sample 2

This is the code needed to achieve that: 

$(document).ready(function(){
    var selector2='#secondpagination';
    $(selector2).monsterPaginate({
        selectPage:15,
        totalPages:56,
        display:10,
        prevText:'Prev',
        nextText:'Next',
        firstText:'First Page',
        lastText:'Last Page',
        callback:onPaginateChange
    });
});


function onPaginateChange(con){
   switch(con.command) {
        case 'nextPage':
            $('.callbackOutput').text("Trigger : "+ con.command)
            break;
        case 'prevPage':
            $('.callbackOutput').text("Trigger : "+ con.command)
            break;
        case 'firstPage':
            $('.callbackOutput').text("Trigger : "+ con.command)
            break;
        case 'lastPage':
            $('.callbackOutput').text("Trigger : "+ con.command)
            break;
        default:
            $('.callbackOutput').text("Trigger : "+ con.selectPage)
    }
}


$('#secondNextPage').click(function() {
    $(selector2).monsterPaginate('nextPage');
})

$('#secondPrevPage').click(function() {
    $(selector2).monsterPaginate('prevPage');
})

$('#secondGoPage').click(function() {
    $(selector2).monsterPaginate('goPage',13);
})

$('#update').click(function() {
    $(selector2).monsterPaginate('update','display',8);
})

Usage

Understands default options that can use for this plugin.
The paginate wrapper Width is auto adjust by init and update, it only work when the div wrapper is visible. 
$('#paginate').monsterPaginate({
    selectPage:1, //page highlight
    totalPages:15, //total pages
    display:5, //total page to display
    nextText:"Next", //Next page text
    prevText:"Prev", //Prev page text
    firstText:"First", //First page text
    lastText:"Last", //Last page text
    showFirstLast:true, //show first last page button; true/false
    autoPrevNext:false, //auto hide prev next page button; true/false
    autoFirstLast:false, //auto hide first last page button; true/false
    callback:'', //call back function
    pageLink:'paginateButton', //follow by default class name
    pageClass:'page',
    prevClass:'prevpage',
    nextClass:'nextpage',
    firstClass:'firstPage',
    lastClass:'lastPage',
    disabledClass:'disabled',
    selected:'selected'
});

//Option can be update later once the same element is init.
$('#paginate').monsterPaginate('update','totalPages',100);
