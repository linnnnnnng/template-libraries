Usage

*Important the form must be init by this format, submit button must stay inside the form wrapper, and submit button ID by deafult is #formSubmit. 
<form id="yourForm">
	<input type="text" validate="email" required id="uEmail" name="formEmail" value="Email" rel="Email" >
    <div id="formSubmit"><a href="">Submit</a></div>
</form>
Understands default options that can use for this plugin. 
$('#yourForm').monsterForm({
    numberOnlyClass:'numberOnly', // class for restrict number only
    submitID:'#formSubmit', //submit button ID
    callback:'' //callback function
});
*Important every input must have attritute 'name' in order for plugin to verify error.
To make the input field auto focus, add attribute rel = 'name' 
<input id="name" name="name" rel="name" type="text" value="name" />
To make the require input field, add attribute required 
<input id="ic" name="ic" rel="ic" required type="text" value="ic" />
To restrict the input field to number only, add class 'numberOnly' 
<input id="contact" name="contact" class="numberOnly" type="text" value="contact" />
To validate email address, add attribute validate='email' 
<input id="email" name="email" validate="email" rel="email" required type="text" value="email" />
To validate input min length, attribute add minlength='5' 
<input id="contact" name="contact" rel="contact" required minlength="3" type="text" value="contact" />
