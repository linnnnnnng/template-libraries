Usage

*Important the form must be initiate by this format, submit button must stay inside the form wrapper, by default the submit ID is #formSubmit. 
<form id="yourForm">
	<input type="text" validate="email" required id="uEmail" name="formEmail" value="Email" placeholder="Email" >
    <div id="formSubmit"><a href="">Submit</a></div>
</form>
Understands default options that can use for this plugin. 
$(document).ready(function(){
    $('#yourForm').monsterForm({
        numberOnlyClass:'numberOnly', // class for restrict number only
        submitID:'#formSubmit', //submit button ID
        callback:'' //callback function
    });
    
    //to reset the form, use the code below after the plugin is initiate
    $('#yourForm').monsterForm('reset');
});
*Important every input must have attritute 'name' in order for plugin to verify error. 
<input name="" />
To make the input field auto on/off focus, add attribute placeholder='your label name' 
<input type="text" id="name" name="name" value="name" placeholder="your label name" />
To make the require input field, add attribute required 
<input type="text" id="ic" name="ic" value="ic" required />
To restrict the input field to number only, add class 'numberOnly' 
<input type="text" id="contact" name="contact" value="contact" class="numberOnly" />
To validate email address, add attribute validate='email' 
<input type="text" id="email" name="email" value="email" required validate="email" />
To validate input file type, add attribute filetype='jpg,jpeg,png' (seperate each file format with comma, no spacing) 
<input type="file" id="uploadfile" name="uploadfile" required filetype="jpg,jpeg,png" />
To validate input file size, add attribute filesize='10000000' (2MB = 2000000) 
<input type="file" id="uploadfile" name="uploadfile" required filesize="1000000" />
To validate input field with custom function 
<input type="text" id="uCustom" name="uCustom" required validate-function="yourFunction" />
