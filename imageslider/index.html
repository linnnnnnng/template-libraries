How to install

First and most important, the jQuery library needs to be included, then include the script tag below in your document and get start! 
<script type='text/javascript' src='js/monster-image-slider.js'></script>
If the draggable option is set to true, the jQuery UI library and Touch Punch needs to be included! 
<script type='text/javascript' src='https://code.jquery.com/ui/1.12.1/jquery-ui.js'></script>
<script type='text/javascript' src='jquery.ui.touch-punch.js'></script>
DOWNLOAD monster-image-slider.js

Sample 1

This is the code needed to achieve this:
Note that the call must be made inside of a $(document).ready() call, or the plugin will not work! 
$(document).ready(function(){
    $('#imgSlider').monsterImageSlider({
        timer:3000,
        draggable:true
    });
});

Sample 2

This is the code needed to achieve that: 
$(document).ready(function(){
    $('#imgSlider2').monsterImageSlider({
        autoShowLeftRight:true,
        autoShowControl:true,
        timer:0
    });
});

Usage

Understands default options that can use for this plugin. 
$('#imgslider').monsterImageSlider({
	draggable:true, //make the slide draggable; true/false
    autoShowControl:false, //auto show left right control; true/false
    autoShowLeftRight:false, //auto show image list control; true/false
    timer:3000, //slide show timer; 0 = off
    selectSlide:1, //show which image slide
    callback:'', //call back function
    imageHolder:".imageList", //follow by default elements
    controlHolder:'.imageControl',
    navHolder:'.nav',
    navLeftHolder:'.leftArrow',
    navRightHolder:'.rightArrow',
    selectedClass:'selected'
});

//Option can be update later once the same element is init.
$('#imgslider').monsterImageSlider('update','timer',5000);

Destroy
